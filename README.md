# 3D-визуализация логических схем

Учебный интерактивный 3D-фрагмент для объяснения принципа работы электронных схем, реализующих простейшие логические функции: `NOT`, `AND`, `OR`.

Проект сделан не как статичный ролик, а как обучающий материал: пользователь видит объемную схему, формулу функции, таблицу истинности, пошаговое объяснение текущего состояния и может проверить себя коротким вопросом по выходу `Y`.

## Запуск

```bash
npm install
npm run dev
```

Локальный адрес:

```text
http://127.0.0.1:5173/
```

## Сборка

```bash
npm run build
```

## Что реализовано

- Объемная 3D-сцена с платой, дорожками, цилиндрическими проводниками, переключателями, транзисторами, корпусом логического элемента, ножками микросхемы, источником питания и светодиодом.
- Визуализация функций `NOT`, `AND`, `OR`.
- Автоматический режим мини-видео с главами и плавными переходами между состояниями.
- Интерактивный режим: выбор функции и переключение входов `A`, `B`.
- Анимация прохождения сигнала частицами по проводникам.
- Формула текущей функции.
- Пошаговое учебное объяснение текущего состояния схемы.
- Таблица истинности с подсветкой активной строки.
- Мини-проверка знаний: пользователь выбирает, чему равен выход `Y`.

## Как записать видео

1. Запустить проект командой `npm run dev`.
2. Открыть `http://127.0.0.1:5173/`.
3. Оставить режим `Видео`.
4. В OBS выбрать запись окна браузера или экрана.
5. Нажать кнопку запуска сначала и записать один полный проход.
6. При необходимости озвучить текст из `docs/video-script.md`.

## Как показать интерактив

1. Переключить режим на `Интерактив`.
2. Выбрать функцию `NOT`, `AND` или `OR`.
3. Нажимать на входы `A` и `B`.
4. Смотреть, как меняются схема, таблица истинности, формула, объяснение и выход `Y`.
5. Ответить на вопрос в блоке `Проверь себя`.
