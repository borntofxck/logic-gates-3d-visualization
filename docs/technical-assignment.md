# Краткое техническое задание

## Тема

Разработка обучающего видеофрагмента трехмерной визуализации «Принцип работы электронных схем, реализующих простейшие логические функции».

## Назначение

Разрабатываемый программный продукт предназначен для использования в учебном процессе колледжа при объяснении базовых логических функций и принципов работы простейших электронных схем.

Материал должен быть не просто видеороликом, а интерактивным обучающим фрагментом, по которому студент может изучить тему, увидеть объемную модель схемы, сопоставить ее с таблицей истинности и проверить понимание результата.

## Цель разработки

Создать браузерную 3D-визуализацию, демонстрирующую работу логических функций `NOT`, `AND`, `OR` через объемные модели электронных компонентов, анимацию прохождения сигнала, учебные пояснения и интерактивную проверку знаний.

## Функциональные требования

1. Отображать объемную 3D-сцену с учебной электронной платой.
2. Показывать источник питания, проводники, переключатели входов, логический элемент и светодиод выхода.
3. Демонстрировать функции `NOT`, `AND`, `OR`.
4. Отображать текущие входы `A`, `B` и выход `Y`.
5. Показывать таблицу истинности и подсвечивать активную строку.
6. Показывать формулу текущей логической функции.
7. Показывать пошаговое объяснение текущего состояния схемы.
8. Реализовать автоматический режим мини-видео с главами.
9. Реализовать интерактивный режим с ручным переключением входов.
10. Реализовать мини-проверку знаний с выбором значения выхода `Y`.
11. Добавить плавные переходы между состояниями схемы.
12. Добавить анимацию прохождения сигнала по проводникам.

## Нефункциональные требования

1. Проект должен запускаться в современном браузере.
2. Интерфейс должен быть понятен студенту без дополнительной инструкции.
3. Визуализация должна быть достаточно объемной, чтобы соответствовать теме трехмерной визуализации.
4. Переходы между состояниями должны быть плавными, без резкого исчезновения схемы.
5. Проект должен запускаться локально через `npm run dev`.
6. Проект должен успешно собираться командой `npm run build`.

## Используемые технологии

- Vite;
- TypeScript;
- Three.js;
- GSAP;
- HTML;
- CSS.

## Ожидаемый результат

Итогом разработки является интерактивный учебный 3D-фрагмент, который можно использовать двумя способами:

1. как готовую демонстрацию для записи обучающего видео;
2. как интерактивный учебный пример для самостоятельной проверки работы логических функций.
